# Newman

https://www.npmjs.com/package/newman

## Requirements ##

```shell
Node.js >= v10
```

## Typescript ##

### Install

NPM:
```shell
npm install -g newman
```

### Build

Run:
```shell
newman run <pathToYourFile> -e <pathToYourEnvironment>
```
